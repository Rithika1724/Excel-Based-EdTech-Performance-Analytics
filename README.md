# 📊 Excel-Based EdTech Performance Analytics

An interactive Microsoft Excel dashboard built to analyze business performance for an Ed-Tech organization. This project transforms raw enrollment, marketing, payment, and trainer data into meaningful business insights using Excel's analytical capabilities.

---

## 📌 Project Overview

This dashboard provides a complete view of student enrollments, revenue generation, marketing effectiveness, trainer performance, and course popularity.

Instead of simply visualizing data, the project focuses on identifying business trends that support better decision-making.

---

## 🎯 Business Objectives

- Monitor overall enrollments and revenue
- Identify the most popular courses
- Evaluate trainer satisfaction
- Analyze city-wise enrollments
- Track monthly revenue trends
- Understand marketing channel performance
- Provide actionable business insights

---

## 💡 Key Insights

- Data Analytics recorded the highest enrollments among all courses.
- Meera achieved the highest trainer satisfaction score.
- Chennai emerged as one of the strongest enrollment contributors.
- Google Ads and Affiliate campaigns generated significant student acquisition.
- Monthly revenue remained stable with moderate seasonal fluctuations.
- A trainer allocation inconsistency was identified for the Data Analytics course, highlighting the importance of data validation.
